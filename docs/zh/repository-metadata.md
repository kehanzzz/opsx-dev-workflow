## 一句话介绍

建议文案：

`从探索到归档，再到分支收尾。一个工作流，明确的阶段门。`

## About 区域

建议突出这 4 点：

- 它是 workflow skill，不是上游工具包镜像
- 直接依赖上游 OpenSpec 和 superpowers
- 强调状态记录和审计链路
- 面向 Codex、Claude Code、Gemini CLI、opencode
